# Zhi Chen | Audit & Advisory Analytics Portfolio

## 📊 Project 1: Analyzing Sales Data in Alteryx
* **Objective:** 
* **Core Technology:** Alteryx Designer (ETL & Data Logic) + Microsoft Power BI (Exception Reporting)

---

### 🔍 1. The Business Problem
> "In traditional outfield audit environments, manually matching high-volume transactional data between subsidiary accounts and general ledgers slows down engagement timelines and introduces significant human error. For a client population exceeding 30,000 transaction rows, traditional Excel searching is inefficient and fails to create a secure audit trail."

### 🛠️ 2. Technical Approach & Methodology
1. **Data Ingestion:** Extracted and parsed multi-currency general ledger text files and subsidiary databases.
2. **Data Cleansing:** Standardized field types, date formats, and removed white spaces using data cleansing tools.
3. **Join Logic:** Configured Join tools to instantly match unique transaction IDs and values across ledger surfaces.
4. **Exception Handling:** Isolated unjoined data fragments using filter logic to automatically populate an audit risk list.
